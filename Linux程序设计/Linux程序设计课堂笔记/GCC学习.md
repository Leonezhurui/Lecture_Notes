# GCC学习

> By Leonezhurui, Software Engineering



Gcc指令的一般格式为：`Gcc [选项] 要编译的文件 [选项] [目标文件]`

| 后缀名    | 所对应的语言                                         |
| --------- | ---------------------------------------------------- |
| -c        | 只是编译不链接，生成目标文件“.o”                     |
| -S        | 只是编译不汇编，生成汇编代码                         |
| -E        | 只进行预编译，不做其他处理                           |
| -g        | 在可执行程序中包含标准调试信息                       |
| -o file   | 把输出文件输出到file里                               |
| -v        | 打印出编译器内部编译各过程的命令行信息和编译器的版本 |
| -I dir    | 在头文件的搜索路径列表中添加dir目录                  |
| -L dir    | 在库文件的搜索路径列表中添加dir目录                  |
| -static   | 链接静态库                                           |
| -llibrary | 连接名为library的库文件                              |
